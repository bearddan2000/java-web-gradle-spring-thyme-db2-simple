# java-web-gradle-spring-thyme-db2-simple

## Description
A POC for springboot web app with thymeleaf support
connecting to a database.

## Tech stack
- java
- hibernate
- gradle
  - springboot
  - thymeleaf
  - db2 connector
- bootstrap
- jquery
- dataTable

## Docker stack
- ibmcom/db2
- gradle:latest

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
